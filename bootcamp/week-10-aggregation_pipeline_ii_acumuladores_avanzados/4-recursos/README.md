# Recursos — Semana 10

**Aggregation Pipeline II — Acumuladores y Transformaciones Avanzadas**

## Documentación Oficial (MongoDB 7.0)

### Acumuladores de $group

- [$first (aggregation)](https://www.mongodb.com/docs/manual/reference/operator/aggregation/first/) — primer valor del grupo
- [$last (aggregation)](https://www.mongodb.com/docs/manual/reference/operator/aggregation/last/) — último valor del grupo
- [$push (aggregation)](https://www.mongodb.com/docs/manual/reference/operator/aggregation/push/) — acumular todos los valores en array
- [$addToSet (aggregation)](https://www.mongodb.com/docs/manual/reference/operator/aggregation/addToSet/) — acumular valores únicos en array

### Transformaciones de documentos

- [$addFields (aggregation)](https://www.mongodb.com/docs/manual/reference/operator/aggregation/addFields/) — agregar campos calculados
- [$cond (aggregation)](https://www.mongodb.com/docs/manual/reference/operator/aggregation/cond/) — expresión condicional ternaria
- [$ifNull (aggregation)](https://www.mongodb.com/docs/manual/reference/operator/aggregation/ifNull/) — valor por defecto para campos nulos/ausentes

### Referencias de Pipeline

- [Aggregation Pipeline Operators](https://www.mongodb.com/docs/manual/reference/operator/aggregation-pipeline/) — listado completo
- [Aggregation Pipeline Optimization](https://www.mongodb.com/docs/manual/core/aggregation-pipeline-optimization/) — mejores prácticas de rendimiento
