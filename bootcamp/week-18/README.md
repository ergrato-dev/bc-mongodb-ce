# Semana 18 — Patrones de Esquema Avanzados II

## Descripción

Continúa el estudio de patrones de modelado de datos con énfasis en documentos
heterogéneos y evolución de esquemas: Polymorphic, Attribute, Schema Versioning
y Document Versioning.

## Objetivos

1. Modelar entidades heterogéneas en una sola colección con el patrón Polymorphic
2. Indexar eficientemente conjuntos arbitrarios de atributos con el patrón Attribute
3. Gestionar la evolución del esquema sin downtime con Schema Versioning
4. Mantener historial de versiones de documentos con Document Versioning
